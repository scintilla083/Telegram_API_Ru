## Что это

**Telegram MTProto Reference RU** — это неофициальный русскоязычный справочник по **Telegram MTProto API**, собранный в удобном формате для изучения, навигации и практической работы.

Проект объединяет в одном месте:

- все основные **методы** MTProto API
- **сигнатуры** методов: параметры -> return type
- **типы, объекты и конструкторы** TL-схемы
- ссылки на **официальную документацию Telegram**
- **HTML-версию** с удобным поиском
- **JSON-экспорт** для автоматической обработки

---

## Что внутри

- **Полный список методов** MTProto API
- **Описание параметров и возвращаемых типов**

- **Ссылки на официальные страницы Telegram Docs**
- **HTML-справочник** для быстрого просмотра в браузере
- **Markdown-файлы** для чтения и редактирования
- **JSON-версия** для своих инструментов, парсеров и индексации

## Почему это удобно

Официальная документация Telegram мощная, но при работе с большим количеством методов и типов не всегда хватает единого и быстро просматриваемого представления.

Этот репозиторий помогает:

- быстрее находить нужные методы
- видеть сигнатуры без лишних переходов
- использовать данные в удобном **HTML/Markdown/JSON** формате
- работать с MTProto как со структурированным справочником, а не как с разрозненными страницами

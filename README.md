# Telegram MTProto Reference RU

Полная русскоязычная справка по **Telegram MTProto API** на основе официальной схемы и индекса методов.

## Что внутри

* **Все методы** MTProto API
* **Сигнатуры**: параметры → return type
* **Типы и объекты**
* **Ссылки** на официальную документацию Telegram
* **HTML-версия** с удобным поиском
* **JSON-экспорт** для автоматической обработки

## Файлы

* `telegram_mtproto_full_reference_ru.html` — HTML-справочник
* `telegram_mtproto_methods_full_ru.md` — методы
* `telegram_mtproto_types_full_ru.md` — типы и объекты
* `telegram_mtproto_reference_ru.json` — JSON-версия
* `telegram_mtproto_reference_ru_bundle.zip` — архив со всеми файлами

## Для кого

Проект подойдёт для:

* разработчиков Telegram client automation
* исследователей MTProto
* авторов userbot/client API инструментов
* тех, кому нужна полная русская документация в одном месте

## Основа

Справочник собран на базе официальной документации Telegram и актуальной TL-схемы.

---

**Telegram MTProto Reference RU** — это попытка сделать Telegram API более понятным, структурированным и удобным для русскоязычных разработчиков.
